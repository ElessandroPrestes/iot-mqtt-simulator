# Glossário

- **MQTT**: Message Queuing Telemetry Transport. Protocolo leve de mensageria IoT.
- **Mosquitto**: O software de broker MQTT em uso.
- **QoS (Quality of Service)**: O projeto usa QoS 1 (at least once delivery).
- **Timeseries**: Tipo especial de collection no MongoDB otimizada para séries temporais (append-only com timestamp).
- **TTL (Time to Live)**: Configuração que apaga leituras após 7 dias.
- **ECharts**: Biblioteca de gráficos da Apache usada no Vue.js.
- **Pinia**: Gerenciador de estado do Vue.js.
