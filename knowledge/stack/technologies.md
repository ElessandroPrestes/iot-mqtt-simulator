# Tecnologias da Stack

## Backend (API & Simulator)
- **Node.js**: Plataforma de execução. API usa CommonJS; Simulator usa ESM.
- **Express**: Framework HTTP da API.
- **Mongoose**: ODM para MongoDB.
- **MQTT.js**: Cliente para o protocolo MQTT.
- **Socket.io**: Servidor WebSocket.
- **Winston**: Sistema de log em formato JSON.
- **Prom-client**: Exportador de métricas para Prometheus.
- **Jest**: Framework de testes Node.js.

## Frontend (Dashboard)
- **Vue 3**: Framework SPA (Composition API / `<script setup>`).
- **Vite**: Bundler e Dev server ultrarrápido.
- **Pinia**: Estado global da UI.
- **TailwindCSS**: Utilitários CSS.
- **ECharts / vue-echarts**: Gráficos de telemetria.
- **Vitest**: Framework de testes unitários Frontend.

## Infra
- **Docker & Docker Compose**: Orquestração local de containers.
- **MongoDB**: Banco de dados NoSQL (v7.0).
- **Eclipse Mosquitto**: MQTT Broker (v2.0).
