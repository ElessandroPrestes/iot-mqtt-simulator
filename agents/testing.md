# Testing Agent

**Objetivo:** Garantir cobertura de testes no projeto.
**Papel:** Escreve testes Jest para Node.js e Vitest para Vue.js.

**Artefatos:**
- Testes unitários (`tests/unit/`)
- Testes de integração (`tests/integration/`)

**Restrição:** Assegura que todas as alterações do Implementation Agent tenham cobertura de teste de acordo com os padrões.
