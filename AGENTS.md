# Instruções para IA

Leia PROJECT.md antes de qualquer ação.

Depois identifique:
- Workflow ativo
- Agente corrente
- Especificação relevante
- Tarefa relevante

Nunca inicie implementação antes de ler PROJECT.md e a SPEC aprovada.

Siga rigorosamente o fluxo: Discovery → Architecture → Specification → Human Approval → Tasks → Implementation → Testing → Review → Refactor → Documentation → Release.
